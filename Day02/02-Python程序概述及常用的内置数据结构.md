#### 4. 循环语句（20分钟）

**while循环**：

*   格式：`while 条件: 循环体`
*   适用于不确定循环次数的情况

**案例讲解**：辗转相除法求最大公约数（例2.6）

```python
a = int(input("输入正整数a: "))
b = int(input("输入正整数b: "))

# 确保a > b
if a < b:
    a, b = b, a

r = a % b
while r != 0:
    a, b = b, r
    r = a % b
print("最大公约数:", b)
```

**for循环**：

*   格式：`for 变量 in 可迭代对象: 循环体`
*   适用于遍历序列（字符串、列表、元组等）

**range()函数**：

*   `range(stop)`：0到stop-1
*   `range(start, stop)`：start到stop-1
*   `range(start, stop, step)`：指定步长

**循环示例**：

```python
# 遍历字符串
for ch in "Python":
    print(ch, end="-")  # P-y-t-h-o-n-

# 遍历列表
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# 使用索引
for i in range(len(fruits)):
    print(i, fruits[i])
```

#### 5. 用户函数（15分钟）

**函数定义**：

*   使用 `def` 关键字
*   格式：`def 函数名(参数列表): 函数体 [return 返回值]`

**案例讲解**：判断素数的函数（例2.7）

```python
def is_prime(n):
    """判断一个数是否为素数"""
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

# 调用函数统计100-200之间的素数个数
count = 0
for i in range(100, 201):
    if is_prime(i):
        count += 1
print("100~200之间素数个数:", count)
```

**函数优点**：

*   代码复用
*   模块化
*   易于测试和维护

#### 6. lambda函数（10分钟）

**概念**：匿名函数，适用于简单操作

**格式**：`lambda 参数: 表达式`

**示例对比**：

```python
# 普通函数
def add(x, y):
    return x + y

# lambda函数
add_lambda = lambda x, y: x + y

print(add(3, 5))        # 8
print(add_lambda(3, 5)) # 8

# 更常见的用法：作为参数传递
pairs = [(1, 'one'), (2, 'two'), (3, 'three'), (4, 'four')]
pairs.sort(key=lambda pair: pair[1])  # 按单词排序
print(pairs)  # [(4, 'four'), (1, 'one'), (3, 'three'), (2, 'two')]
```

### 第三课时：列表与元组（2.2.1-2.2.2）

#### 1. 导入（5分钟）

#### 2. 列表的创建（15分钟）

**四种创建方式**：

1.  **中括号直接创建**：

```python
list1 = ['S1', '张三', '男', 20, '计算机系']
```

1.  **list()函数转换**：

```python
list2 = list('Python')  # ['P', 'y', 't', 'h', 'o', 'n']
```

1.  **range()创建整数列表**：

```python
list3 = list(range(1, 10, 2))  # [1, 3, 5, 7, 9]
```

1.  **列表推导式**（重点）：

```python
# 基本形式
squares = [x**2 for x in range(10)]  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 带条件的推导式
even_squares = [x**2 for x in range(10) if x % 2 == 0]  # [0, 4, 16, 36, 64]
```

**列表的特点**：

*   有序
*   元素可以不同类型
*   可变（可增删改）

#### 3. 列表的基本操作（25分钟）

**读取元素**：

```python
lst = ['S1', '张三', '男', 20, '计算机系']
print(lst[0])      # S1
print(lst[-1])     # 计算机系

# 遍历
for item in lst:
    print(item, end=' ')
```

**添加元素**：

```python
# append() 尾部追加
lst.append('山东青岛')

# insert() 指定位置插入
lst.insert(3, '团员')
```

**删除元素**：

```python
# remove() 按值删除
lst.remove('计算机系')

# pop() 按下标删除（返回被删除的元素）
deleted = lst.pop(3)

# del 切片删除
del lst[2:4]

# clear() 清空
lst.clear()
```

**修改元素**：

```python
lst[1] = '李四'  # 直接赋值修改
```

**列表嵌套**：

```python
# 3×3矩阵
matrix = [[12, -5, 3],
          [0, 14, -11],
          [23, -45, 8]]

# 访问元素
print(matrix[1][2])  # -11
```

**排序操作**：

```python
nums = [3, 1, 4, 1, 5, 9, 2]
nums.sort()              # 升序 [1, 1, 2, 3, 4, 5, 9]
nums.sort(reverse=True)  # 降序 [9, 5, 4, 3, 2, 1, 1]
nums.reverse()           # 反转

# 获取长度
print(len(nums))
```

#### 4. 元组的创建（10分钟）

**概念**：元组是不可变的列表

**创建方式**：


#### 5. 元组的操作（10分钟）

**读取元素**（与列表相同）：

```python
tup = ('S3', '许文秀', '女', 20, '计算机系')
print(tup[1])      # 许文秀
print(tup[-1])     # 计算机系
```

**元组不可变性的验证**：

```python

```

**元组合并**：

```python

tup2 = (4, 5, 6)

```

#### 6. 列表与元组的对比（5分钟）

| 特性   | 列表       | 元组             |
| ---- | -------- | -------------- |
| 可变性  | 可变       | 不可变            |
| 语法   | `[ ]`    | `( )`          |
| 性能   | 较慢       | 较快（Python会优化）  |
| 适用场景 | 需要修改的数据集 | 不需要修改的数据集，字典的键 |

### 第四课时：字典与集合（2.2.3-2.2.4）

#### 1. 导入（5分钟）

**情景引入**：

#### 2. 字典的创建（10分钟）

**概念**：键值对的集合，键唯一且不可变

**创建方式**：

```python
# 花括号创建
student = {
    '学号': 'S1',
    '姓名': '许文秀',
    '性别': '女',

empty_dict = {}
empty_dict2 = dict()
```

**键的要求**：

*   键必须唯一（重复会覆盖）
*   键必须不可变（数字、字符串、元组可作为键，列表不行）

#### 3. 字典的基本操作（15分钟）

**读取值**：

```python
print(student['姓名'])      # 许文秀
print(student.get('年龄'))  # 20（推荐，键不存在返回None）
```

**添加元素**：

```python
student['籍贯'] = '广西北海'
```

**修改元素**：

```python
student['年龄'] = 22
```

**删除元素**：

```python
del student['性别']
```

**判断键是否存在**：

```python
if '学号' in student:
    print("学号存在")
```

**遍历字典**：

```python
# 遍历键
for key in student.keys():
    print(key, student[key])

# 遍历值
for value in student.values():
    print(value)

# 遍历键值对
for key, value in student.items():
    print(key, ":", value)
```

#### 4. 字典常用方法（10分钟）

**clear()**：清空字典

```python
student.clear()  # {}
```

**update()**：合并字典

```python
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}
dict1.update(dict2)  # {'a': 1, 'b': 3, 'c': 4}
```

**get()**：安全获取值

```python
# 键不存在时返回默认值
score = student.get('成绩', 0)  # 如果'成绩'不存在，返回0
```

**items()、keys()、values()**：

```python
print(student.items())   # 所有键值对
print(student.keys())    # 所有键
print(student.values())  # 所有值
```

#### 5. 集合的创建（10分钟）

**概念**：无序且不重复的元素集合

**创建方式**：

```python
# 花括号
set1 = {1, 2, 3, 4, 5}

# set()函数
set2 = set([1, 2, 2, 3, 3, 4])  # {1, 2, 3, 4} 自动去重
set3 = set("hello")  # {'h', 'e', 'l', 'o'}

# 空集合（注意不能直接用{}，那是空字典）
empty_set = set()
```

**去重特性**：

```python
numbers = [23, 32, -4, 56, -4, 23, -4]
unique_numbers = set(numbers)  # {32, 56, -4, 23}
```

#### 6. 集合的基本操作（10分钟）

**添加元素**：

```python
set1 = {1, 2, 3}
set1.add(4)      # {1, 2, 3, 4}
```

**删除元素**：

```python
set1.remove(2)   # 元素不存在会报错
set1.discard(5)  # 元素不存在不会报错
```

**随机弹出元素**：

```python
popped = set1.pop()  # 随机弹出一个元素
```

**遍历集合**（无序，所以顺序不确定）：

```python
for x in set1:
    print(x, end=' ')
```

#### 7. 集合运算（10分钟）

**示例集合**：

```python
se1 = {'b', 'a', 'd', 'e', 'c'}
se2 = {'p', 'r', 'b', 'o', 't', 'e', 'c'}
```

**并集**（| 或 union()）：

```python
print(se1 | se2)  # {'a', 'b', 'c', 'd', 'e', 'o', 'p', 'r', 't'}
```

**交集**（& 或 intersection()）：

```python
print(se1 & se2)  # {'b', 'c', 'e'}
```

**差集**（- 或 difference()）：

```python
print(se1 - se2)  # 在se1但不在se2中：{'a', 'd'}
print(se2 - se1)  # 在se2但不在se1中：{'o', 'p', 'r', 't'}
```

**对称差集**（^ 或 symmetric\_difference()）：

```python
print(se1 ^ se2)  # 并集减去交集：{'a', 'd', 'o', 'p', 'r', 't'}
```

#### 8. 四种数据结构的对比总结（5分钟）

| 特性    | 列表   | 元组    | 字典         | 集合      |
| ----- | ---- | ----- | ---------- | ------- |
| 有序性   | 有序   | 有序    | 无序（3.7+有序） | 无序      |
| 可变性   | 可变   | 不可变   | 可变         | 可变      |
| 元素唯一性 | 可重复  | 可重复   | 键唯一        | 唯一      |
| 访问方式  | 下标   | 下标    | 键          | 不可直接访问  |
| 适用场景  | 有序序列 | 不可变序列 | 键值映射       | 去重、集合运算 |

### 实验目的

*   熟练使用Python的基本数据类型和数据结构
*   能够编写完整的Python程序解决简单问题

### 实验题目

#### 基础题（必做）

1.  **字符串处理**：输入一个句子，统计其中每个单词出现的次数
2.  **列表操作**：生成20个随机整数，计算平均值、最大值、最小值
3.  **字典应用**：建立一个电话簿，实现添加、删除、查找、显示所有联系人功能

#### 提高题（选做）

1.  **学生成绩管理系统**：
    *   使用字典存储学生信息（学号、姓名、三门课成绩）
    *   实现添加学生、删除学生、修改成绩、计算平均分、按平均分排序等功能
2.  **集合应用**：模拟两个班级的学生，找出既在一班又在二班的学生、只在一班的学生等

### 实验要求

*   代码规范（缩进、注释、变量命名）
*   每个题目独立保存为.py文件
*   提交实验报告（代码+运行结果截图+遇到的问题及解决）

## 五、课后作业

### 理论作业

1.  简述列表、元组、字典、集合的区别和适用场景
2.  解释为什么字典的键必须是不可变类型
3.  写出下列代码的输出结果（考察切片和列表推导式）

### 编程作业

1.  编写程序，输入一个字符串，输出反转后的字符串（分别用切片和循环实现）
2.  生成斐波那契数列的前20项，存入列表并输出
3.  统计一篇文章（可从文件读取）中每个字母出现的次数，忽略大小写
4.  模拟购物车系统：使用列表存储商品信息，支持添加、删除、修改数量、计算总价

## 七、教学反思与建议

### 常见问题及对策

1.  **缩进错误**：强调Python对缩进的严格要求，建议使用编辑器自动缩进功能
2.  **可变/不可变混淆**：多用图示说明内存变化，帮助学生理解
3.  **列表推导式理解困难**：从普通循环逐步过渡到推导式
4.  **字典键的要求**：用反例说明为什么列表不能作为键

### 教学建议

*   多安排课堂练习，及时巩固
*   鼓励学生互相review代码，发现错误
*   将知识点与数据挖掘后续内容关联，增强学习动机
